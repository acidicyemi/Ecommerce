## About Ecommerce

This is a sample Ecommerce project base on laravel 5.6


## Learning Setup

Simple Clone the project
### Install via composer
 Composer install

 ### Edit .env file
 Input your enviroment file create the related database

 ### run migration
 php artisan migrate

### Seed to database
php artisan db:seed

### server the project
php artisan serve
